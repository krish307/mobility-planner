# Sustainable Mobility Planner (Streamlit)

This repository contains `pravesh.py` — a small Streamlit app to recommend eco-friendly transport options.

Deploy to Streamlit Cloud:

1. Create a new GitHub repository and push this folder's contents to it.
2. Go to https://share.streamlit.io and sign in with GitHub.
3. Click "New app", select your repository, branch (usually `main`), and `pravesh.py` as the main file.
4. Click Deploy — Streamlit Cloud will install dependencies from `requirements.txt` and launch the app.

Local test:

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1   # Windows PowerShell
pip install -r requirements.txt
streamlit run pravesh.py
```
